# Website

React + TypeScript + Vite + TailwindCSS static SPA for the Family Function
Photo Finder.

## Develop

```bash
npm install
npm run dev
```

Vite serves `public/data/people.json` directly, so the dev server reflects
whatever the processor wrote.

## Build

```bash
npm run build
```

Outputs static files in `dist/`. Drop those into:

- **GitHub Pages**: push `dist/` to a `gh-pages` branch.
- **Cloudflare Pages**: build command `npm run build`, output directory `dist`.
- **Netlify**: drag `dist/` into the dashboard.

## How data flows in

`public/data/` is generated by the processor:

```bash
cd ../processor
uv run photo-finder deploy
```

That command copies `people.json`, `faces/`, and `photo-thumbnails/` into this
project's `public/data/` directory.

## Routing

Uses hash routing (`#/p/:id`) so the bundle works under any static host without
server-side rewrites.
